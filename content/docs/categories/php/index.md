## Fonctionnement
### Lecture
Symfony lit un fichier `.md`, CommonMark génère le HTML, puis Twig applique les styles.
### Conversion
Les titres génèrent des ancres automatiques et alimentent la table des matières.
### Affichage
Le TOC doit rester collé à droite sur desktop et repasser en pleine largeur sur mobile.
